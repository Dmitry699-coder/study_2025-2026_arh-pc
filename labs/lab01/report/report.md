Ход работы

    Определение домашнего каталога
    С помощью команды pwd был определён полный путь к домашнему каталогу:
    text

/home/dmitry

    Навигация по файловой системе

        Переход в подкаталог Документы с использованием относительного пути.

        Переход в каталог /usr/local с использованием абсолютного пути.

    Использование команд cd - и cd ..
    Команда cd - позволила вернуться в последний посещённый каталог, а cd .. — подняться на уровень выше в иерархии файловой системы.

    Просмотр содержимого каталогов
    С помощью команды ls был выведен список файлов домашнего каталога. Результаты совпали с отображением в графическом файловом менеджере.

    Создание каталогов и файлов

        Созданы каталоги parentdir, newdir, а также вложенные структуры (dir1/dir2).

        Созданы текстовые файлы с помощью команды touch.

        Использованы ключи -p для создания цепочки вложенных каталогов и -i для интерактивного подтверждения действий.

    Управление файлами

        Удаление файлов с расширением .txt с подтверждением (rm -i).

        Рекурсивное удаление каталогов (rm -R).

        Копирование и перемещение файлов с помощью команд cp и mv.

        Переименование файлов и каталогов.

    Работа с содержимым файлов
    Команда cat использована для просмотра содержимого системного файла /etc/hosts и созданных текстовых файлов.

    Самостоятельная работа

        Повторены основные операции: создание каталогов temp и labs, создание файлов text1.txt, text2.txt, text3.txt.

        В файлы записаны личные данные: имя, фамилия, номер группы.

        Файлы скопированы, переименованы и перемещены в соответствующие подкаталоги.

        Все созданные файлы и каталоги удалены командой rm -r.
